# 每日阅读 - 复古电报与报纸阅读器 (Retro Telegraph & Newspaper Reader)

> 这是一个极具个性的复古拟物风（Skeuomorphic）交互式网页应用，结合了**复古蒸汽朋克电报机**与**老式印刷报纸**的视觉与交互体验。

**在线访问**: [https://like-455.github.io/retro-telegraph-reader](https://like-455.github.io/retro-telegraph-reader/)


## ✨ 核心特性

- 📻 **复古拟物设计**：精致的铜质电报机、逼真的发报按键、动态指针、玻璃质感拟物面板，带来沉浸式体验。
- 📰 **排版老报纸**：自动将发报或输入的内容排版为复古风格的老式印刷报纸，带有动态盖印与做旧纸张纹理。
- ⌨️ **支持中英文发报**：
  - 支持直接敲击摩尔斯电码发报。
  - 支持键盘输入，且完美支持中文输入法（IME 优化），输入中文会自动将其转换为拼音，再转换为莫尔斯电码进行“发报”播放。
  - 拥有退格（Backspace）删除已打印字符的功能。
- 📢 **真实电报音效**：高保真模拟电报滴答声，配合发报动作实时发出。
- 📰 **今日简报异步加载**：自动调用新闻 API，将最新的 15 条新闻排版成 15 篇独立的复古报纸简报。
- 📸 **一键截图分享**：支持将生成的报纸页面一键保存为精美图片。

## 🛠️ 本地运行

1. 克隆或下载本项目至本地。
2. 在项目根目录下打开终端，安装依赖：
   ```bash
   npm install
   ```
3. 启动本地开发服务器：
   ```bash
   npm run dev
   ```
4. 在浏览器中打开提示的本地链接（默认：`http://localhost:3000`）。
